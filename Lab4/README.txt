Lab 4: Paint application

-------------
DESCRIPTION

In this lab, the user follows the instructions given and inputs a number for the x and y starting coordinates and proceeds to play with the wasdqezc keys in order to direct the pixels movements.

-------------

FILES

-
lab4_part1.asm
lab4_part2.asm

This file includes the assembly code of the lab.

INSTRUCTIONS

This program is intended to be run using the RISCV Assembler and Runtime Simulator
(RARS). Enter the test case as a program argument and run using RARS.
Open the lab4_v2.asm file as well as the "Bitmap Display" and "Keyboard and Display MMIO Simulator" and set the Bitmap Display to 8, 8, 256, 256, 0x10008000(gp). Then assemble the file, connect both of the tools to "Connect to Program" and run it. Insert the numbers 0-31 for x and y and proceed typing on the keyboard using wasdqezc to see the red pixel move.
